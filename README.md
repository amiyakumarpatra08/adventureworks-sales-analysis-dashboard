# AdventureWorks Sales Analysis Dashboard

## Project Overview
This project analyzes AdventureWorks sales data using SQL, Power BI, and Tableau to generate business insights and interactive dashboards.

## Objectives
- Analyze sales performance
- Identify top products and regions
- Track revenue trends
- Create interactive dashboards
- Support business decision-making

## Tools Used
- SQL – Data extraction and analysis
- Power BI – Interactive dashboard development
- Tableau – Data visualization
- Excel – Data preparation

## Key Features
- Sales trend analysis
- Regional performance analysis
- Product performance insights
- Customer analysis dashboard
- KPI tracking

## Files Included
- AdventureWorks Sales Report.sql → SQL queries
- power bi AW Project.pbix → Power BI Dashboard
- Tableau AW project new._.twbx → Tableau Dashboard
